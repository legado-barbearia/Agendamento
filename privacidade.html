Uso de dados no agendamento

Esta página explica de forma simples como os dados informados no site são usados pela Legado Barbearia.

Quais dados são solicitados

O formulário pode solicitar nome, número de WhatsApp, serviço, data, horário, profissional escolhido e observações relacionadas ao atendimento.

Para que os dados são usados

As informações servem exclusivamente para registrar, organizar, confirmar, reagendar ou cancelar atendimentos e para manter contato com o cliente sobre o horário reservado.

Armazenamento nesta versão

Enquanto o sistema não estiver conectado ao Supabase, os dados ficam armazenados localmente no navegador utilizado. Eles não são sincronizados automaticamente entre aparelhos.

Compartilhamento

A Legado Barbearia não deve vender ou compartilhar os dados dos clientes para publicidade de terceiros. O WhatsApp é aberto somente quando o próprio usuário ou administrador escolhe enviar uma mensagem.

Correção e exclusão

O cliente pode solicitar correção ou exclusão de seus dados entrando em contato pelo WhatsApp da barbearia. O administrador também pode excluir registros pelo painel.

Segurança futura

Quando a integração online for ativada, o banco de dados deverá utilizar autenticação, regras de acesso e políticas de segurança para limitar o acesso às informações.
